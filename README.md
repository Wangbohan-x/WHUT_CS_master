# 武汉理工大学计算机硕士论文LaTeX模板

参考  [wutthesis2024](https://github.com/kon9chun/wutthesis2024/tree/main) 模板，根据武汉理工大学计算机学院硕士毕业论文要求，对其进行调整

# 编译工具

[TexLive2025](https://tug.org/texlive/windows.html)+ [VSCode](https://code.visualstudio.com/)(LatexWorkshop)

1、下载TexLive2025，并将其添加到系统环境路径中

2、下载VSCode并在拓展商店中下载LatexWorkshop

3、模板自带.vscode/settings.json文件，可直接进行编译，也可自行调整

# 使用方式

可直接把setting.json文件放到同目录的.vscode文件夹下，直接点击右上角编译按钮。

模板使用详见main.pdf
